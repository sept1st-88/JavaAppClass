//			프로그램이 실행 중 어떤 원인에 의해서 오작동을 하거나 비정상적으로 종료되는
//			경우가 있다 이러한 결과를 초래하는 원인을 프래그램 에러 또는 오류라고 한다
//			이를 발생시점에 따라 컴파일 에러와 런타임 에러로 나눌 수 있다
//	컴파일 에러(compile-time error)?
//			컴파일 할 때 발생하는 에러
//			(실행과는 무관한. 코드 실수.)
//	런타임 에러(runtime error)?
//			컴파일도 잘 되고 실행도 잘 되지만 의도한 것과 다르게 동작하는 것
//			(코드 수행시점(메인) - 수행 시점에서 나타나는 에러)
//	논리적 에러(logical error)?
//		컴파일도 잘 되고 실행도 잘 되지만 의도한 것과 다르게 동작하는 것
//	(컴파일을 에러없이 성공적으로 마쳤다고 해서 프로그램의 실행 시에도 에러가 발생하지 않는 것은 아니다)
//	
//	자바에서는 실행 시 발생할 수 있는 프로그램 오류를 에러(error)와 예외(exception)
//	두 가지로 구분하였다
//	에러?
//			프로그램 코드에 의해서 수습될 수 없는 심각한 오류
//			예: 컴퓨터 선이 빠지거나, 전기가 나가 강제 종료 된 것 (물리적인 것)
//	예외?
//			프로그램 코드에 의해서 수습될 수 있는 다소 미약한 오류
